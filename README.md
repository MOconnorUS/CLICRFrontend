# CLICR Frontend

Simplified prototype implementation of CLICR mobile app frontend

## Features
- Venue-specific occupancy tracking via WebSockets
- Authentication flow
- Real-time live count, people entered, people exited updates
- Real-time interaction with +/- buttons to control people entered and people exited

## Tech Stack
- React Native
- Expo
- Supabase

## Notes
This project is configured for local use only  
The UI for this project is not the most refined as functionality was the core focus  
Environment variables are intentionally omitted
